# 奇思岛 · 产品与玩法设计

## 一、产品定位

一款面向 6-8 岁孩子的跨平台学习游戏。整体以"奇思岛"为世界观容器——一片漂浮在云海之上的
群岛，每座岛屿对应一个学科主题。孩子作为"小小拾光人"（暂定），乘小船在群岛间穿梭，点亮
每座岛上的知识。

**核心情绪目标**：**"我在探索一个属于我的小世界，顺便学到了东西。"**
学习目标嵌入到玩法里，而不是贴在玩法外。

## 二、世界观与主线

- 传说很久以前，天地间有一位叫仓颉的智者造出了汉字，天雨粟、鬼夜哭。此后，各种"知识
  的火种"散落到云海之上的一片群岛里，等待被点亮。
- 孩子来到奇思岛，从**字之岛**（第一章）开始，一岛一岛地探索、拾取、点亮。
- 每点亮一样东西，岛屿的一小块就会从水墨灰变成暖色，最终整座岛"活"过来。

## 三、章节规划

| 章节 | 主题 | 核心学习目标 | 玩法关键词 |
|------|------|----------------|-----------|
| 字之岛（首章） | 汉字造字法 | 象形/会意/形声/指事 | 挖掘、拼合、字形演变 |
| 数之岛 | 数感与应用题 | 加减乘除、分数、估算 | 经营、配料、找零 |
| 机关岛 | 物理与逻辑 | 重力、杠杆、齿轮、电路 | 沙盒搭建、连锁反应 |
| 故事岛 | 阅读与表达 | 词汇、句式、想象力 | 故事共创、填空续写 |

首版只做**字之岛**，做深做透，其他岛留清晰接口，后续按同样模板扩展。

## 四、字之岛核心循环

一局约 3 分钟，四步：

1. **勘探**：孩子进入一个水墨场景（河岸/山林/村落/田野），点击可疑之处（土堆、竹丛、石缝）
   触发挖掘小游戏——用手指刷开泥土，露出甲骨文碎片。
2. **拼合**：把 2-4 块碎片拖到轮廓里，拼成一个完整的甲骨文字形。轮廓有轻微磁吸辅助。
3. **演变**：一段 10-15 秒的动画演示这个字从甲骨文 → 金文 → 小篆 → 楷书的形变过程，
   配 TTS 语音讲述字源故事（如"日"："古人看到天上的太阳，就画了一个圆圈，中间点一点，
   代表太阳的光"）。
4. **应用**：一个巩固小挑战——三选一识别、或将该字拖入一句短语的空格（"太______出来了"）。
   答对后，场景中对应的物件（如天上的太阳）从灰变亮，岛屿点亮度 +1。

完成一个场景（5 字）后，进入**小结关**：把这 5 个字组成一句短语或短诗，如"日出山间，
鱼游水中"，触发一段整场景"点亮"的仪式感动画。

## 五、首章内容规模

- **章节名**：字之岛 · 象形篇《万物有形》
- **20 个字**（详见 chapter-01-字之岛.md）：日、月、山、水、火、木、人、口、目、耳、
  手、足、鱼、鸟、马、牛、羊、门、雨、田
- **4 个场景 × 每场景 5 字**：河岸、山林、村落、田野
- **1 个 Boss 关**：把 20 字组成主题短诗

## 六、家长端

首版做**极简**的家长视图（一个入口从主菜单进入，密码/长按解锁）：

- 本周孩子点亮了多少个字
- 最喜欢的一个字（回访次数最多）
- 建议的亲子共读小片段（"孩子今天学了'休'，可以指着爸爸靠在树上的样子说给他听"）

不做付费、不做积分、不做排行榜。以"陪伴"而非"竞赛"为基调。

## 七、美术与音效基调

- **视觉**：绘本水墨风。宣纸底、淡墨勾边、暖色（赭石、藤黄、朱砂）点染，避免高饱和荧光色。
- **UI**：极少文字，图标为主。按钮圆润，反馈以水墨晕开的动画为主。
- **音效**：毛笔勾画的沙沙声、竹简翻动、远处水声、鸟鸣。避免电子音效。
- **BGM**：古琴/箫/木鱼为主的轻氛围乐，循环但不抢戏。
- **TTS**：Edge / Azure 中文儿童声（xiaoxiao、xiaoyi 试听后择优），语速略慢。

## 八、技术要点

- **引擎**：Flutter 3.22+ / Dart 3.4+，Material 3。
- **状态**：`provider` 轻量注入 + `ChangeNotifier`；后续视规模再决定是否升级到 Riverpod。
- **数据**：字库以 JSON 结构描述（字形、拼音、字源故事、演变序列、场景归属、应用题目），
  运行时通过 `rootBundle` 加载，字形/语音资源用 `assets/` 打包。
- **存档**：`shared_preferences` 本地存档（点亮的字 id 集合 + 每字回访计数），后续接云同步。
- **音频**：`audioplayers` 播放 TTS mp3，同一时刻只播一段。
- **构建目标**：iOS / Android / Web；桌面端 (Windows/macOS/Linux) 作为副产物顺手支持。
- **性能预算**：60fps 稳定，冷启动到可玩 < 5 秒，首个场景加载 < 3 秒。
